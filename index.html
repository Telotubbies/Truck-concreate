Step 3 — Motion Analysis

Farneback (แนะนำ) Lucas-Kanade RAFT (GPU) ไม่ต้อง Train ✅

Step 4 — Decision Logic
⚖️ ตัดสิน: หมุน / ไม่หมุน
คำนวณ "Rotational Score" จาก flow vectors — ดูว่า distribution ของ angle มีลักษณะหมุนรอบจุดศูนย์กลางหรือไม่
🟢 ROTATING
score ≥ threshold
🔴 NOT ROTATING
score < threshold
Rotation Detection Logic
①
คำนวณ flow magnitude ต่อ pixel — ถ้า mag เฉลี่ยต่ำมาก = ไม่มีการเคลื่อนที่เลย (หยุดนิ่ง)
②
แบ่ง drum เป็น grid (เช่น 4×4) คำนวณ mean angle ในแต่ละ cell — drum หมุน = flow ใน cell ซ้ายชี้ลง, ขวาชี้ขึ้น (หรือกลับ)
③
คำนวณ Curl / Rotational Component ของ vector field — ค่า curl สูง = มีการหมุนวน
④
Average ผลใน N frames (เช่น 5-10 frames) ก่อนตัดสิน เพื่อลด false positive จาก noise หรือการสั่นของรถ
⑤
ตั้ง threshold จาก validation data — เริ่มจาก mean_magnitude > 1.5 px/frame และ curl_score > 0.3

Code Structure
PYTHON — drum_rotation_detector.py 
# ==============================
# Drum Rotation Detector Pipeline
# ==============================

import cv2
import numpy as np
from ultralytics import YOLO
from collections import deque

# --- Config ---
MODEL_PATH = "best.pt"          # YOLOv8 trained model
DRUM_CLASS_ID = 0               # class index ของ drum
MAG_THRESHOLD = 1.5             # pixel/frame
SMOOTHING_FRAMES = 8            # temporal smoothing

class DrumRotationDetector:
    def __init__(self):
        self.model = YOLO(MODEL_PATH)
        self.prev_gray = None
        self.score_buffer = deque(maxlen=SMOOTHING_FRAMES)

    def detect_drum(self, frame):
        """Step 1: YOLO detect drum bbox"""
        results = self.model(frame, verbose=False)[0]
        for box in results.boxes:
            if int(box.cls) == DRUM_CLASS_ID:
                x1,y1,x2,y2 = map(int, box.xyxy[0])
                return (x1, y1, x2, y2)
        return None

    def crop_drum(self, frame, bbox):
        """Step 2: Crop ROI with padding"""
        x1,y1,x2,y2 = bbox
        pad = 10
        h, w = frame.shape[:2]
        x1,y1 = max(0,x1-pad), max(0,y1-pad)
        x2,y2 = min(w,x2+pad), min(h,y2+pad)
        return frame[y1:y2, x1:x2]

    def calc_rotation_score(self, curr_crop):
        """Step 3-4: Optical Flow + Rotation Score"""
        curr_gray = cv2.cvtColor(curr_crop, cv2.COLOR_BGR2GRAY)
        if self.prev_gray is None:
            self.prev_gray = curr_gray
            return 0.0
        
        # Resize ให้ match ถ้า bbox เปลี่ยน
        target_h, target_w = self.prev_gray.shape
        curr_resized = cv2.resize(curr_gray, (target_w, target_h))
        
        flow = cv2.calcOpticalFlowFarneback(
            self.prev_gray, curr_resized, None,
            0.5, 3, 15, 3, 5, 1.2, 0
        )
        mag, _ = cv2.cartToPolar(flow[...,0], flow[...,1])
        self.prev_gray = curr_resized
        return float(np.mean(mag))

    def process_frame(self, frame):
        """Full pipeline per frame"""
        bbox = self.detect_drum(frame)
        if bbox is None:
            return "NO_DRUM", frame

        crop = self.crop_drum(frame, bbox)
        score = self.calc_rotation_score(crop)
        self.score_buffer.append(score)

        # Smooth decision
        avg_score = np.mean(self.score_buffer)
        status = "ROTATING" if avg_score > MAG_THRESHOLD else "STOPPED"
        
        # Draw
        color = (0,255,0) if status=="ROTATING" else (0,0,255)
        cv2.rectangle(frame, bbox[:2], bbox[2:], color, 2)
        cv2.putText(frame, f"{status} ({avg_score:.2f})",
                    (bbox[0], bbox[1]-10),
                    cv2.FONT_HERSHEY_SIMPLEX, 0.6, color, 2)
        return status, frame
Tips & Pitfalls
📦
Dataset สำหรับ YOLO
ต้องการ 200-500 ภาพขั้นต่ำ แนะนำใช้ Roboflow annotate + augment — label เฉพาะ "drum" ไม่ต้อง label ทั้งรถ จะให้ผลดีกว่า
🚗
Camera Motion Problem
ถ้ากล้องเคลื่อน (ติดรถ) Optical Flow จะ noisy — ต้องทำ Camera Stabilization ก่อน หรือใช้ Background Subtraction ช่วย
🔦
แสงและสีของโม่
โม่ที่สีเดียวกันหมด (เทาล้วน) Optical Flow จะทำงานได้แย่ลง — ใช้ edge ช่วยหรือ combine กับ texture analysis เพิ่มเติม
⏱️
Temporal Smoothing
อย่าตัดสินจาก frame เดียว — buffer 5-10 frames แล้วค่อย output สถานะ จะลด false alarm จากการสั่นของรถหรือ noise ได้มาก
🎯
Threshold Tuning
ค่า threshold ขึ้นกับ resolution, FPS, และระยะห่างกล้อง — ต้อง tune จาก validation video จริง ไม่ควรใช้ค่า hardcode
💻
Hardware Estimate
CPU เพียงพอสำหรับ 1 กล้อง (YOLOv8n + Farneback) ที่ ~15-20 FPS — ถ้า multi-camera แนะนำ GPU หรือ Jetson Nano
Training Steps (YOLO)
BASH — Training Commands 
# 1. Install
pip install ultralytics roboflow

# 2. Download dataset จาก Roboflow (หลัง annotate แล้ว)
from roboflow import Roboflow
rf = Roboflow(api_key="YOUR_KEY")
project = rf.workspace("your-workspace").project("drum-detection")
dataset = project.version(1).download("yolov8")

# 3. Train
yolo train model=yolov8n.pt data=dataset/data.yaml \
    epochs=100 imgsz=640 batch=16 \
    name=drum_detector patience=20

# 4. Validate
yolo val model=runs/detect/drum_detector/weights/best.pt \
    data=dataset/data.yaml

# 5. Export สำหรับ Deploy (optional)
yolo export model=best.pt format=onnx  # สำหรับ CPU deploy
